# git 
 
##  git branch usage
 
  git branch   //列出所有的分支
  git branch <branch> //创建名为<branch>的分支，但是不会切换过去
  git branch -d <branch>  //删除指定分支，这是一个“安全”操作，git会阻止你删除包含未合并更改的分支。
  git branch -D <branch>  //强制删除分支
  git branch -m <branch> //重新命名当前分支
